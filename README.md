## CSCI321 Project - UOW Alumni App

React Native
  - Android
  - iOS

## Requirements

- Node.js
- React Native
- React Native CLI
- Expo


### Installing

How to get react native setup:

https://facebook.github.io/react-native/docs/getting-started.html

All dependencies listed in package.json

App:

Clone git repo.
Inside directory:

```
npm install
```
```
npm start
```
